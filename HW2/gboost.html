Project 2 Part 1: Gradient Boosting

Intro to Gradient Boosting

Gradient Boosting is a machine learning method mainly used for regression and classification. The method involves using an initial model to make predictions, then taking the residuals from those predictions to improve the model. This is done by building a separate model, or models, that predict the errors from the first model. This is iterated through a number of boosting steps, which tells us how many times we will predict the residuals, using a set learning rate to achieve better predictions with each iteration. It is assumed that as we add more models and boosting steps, our predictions for the residuals will decrease.

Gradient Boosting Implementation

In this project, we use Gradient Boosting to improve our predictions on the Lowess Class. Here is my Lowess class, designed for locally weighted regression tasks, creating a smoothed line for visualizing the relationship between two variables. Because the Lowess class approximates the least squares line nearly continuously, it is a relatively strong model on its own; however, the GradientBooster class implemented here is used to further reduce mean squared error, yielding a more accurate model overall.
